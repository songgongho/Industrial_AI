# 고객사 추가 데이터 요청안

이 문서는 PRESS 공정 불량 분석, 품질 추세 분석, 설비 이상 탐지 모델 학습에 필요한 추가 데이터 요청 항목을 정리한 것입니다.

## 1. 현재 확보된 데이터로 가능한 분석

현재 RAW 파일 기준으로 다음 분석은 가능합니다.

- 고객사 불량 유형 마스터 정리
- 연도별/기간별 품질 현황 비교
- PRESS 공정 로그의 압력·온도·진공 변수 상관관계
- 공정 이상과 품질 악화의 간접 연관성 확인

## 2. 꼭 요청해야 하는 데이터

| 우선순위 | 데이터 | 필요한 이유 |
|---|---|---|
| 필수 | 일별 품질 현황 | 급격한 품질 저하 시점, 일중 패턴, 알람과의 정합 분석 |
| 필수 | 주간 품질 현황 | 주간 변동, 교대/운영 변경 영향, 유지보수 전후 비교 |
| 필수 | 월별 품질 현황 | 장기 추세, 시즌성, 개선 효과 확인 |
| 필수 | 설비 가동/비가동 현황 | 실제 생산 시간 대비 품질 영향, 비가동 원인 파악 |
| 필수 | 공정별 설비 가동율 | PRESS 포함 공정 병목과 라인 편차 분석 |
| 필수 | PRESS 공정 알람 내역 | 이상 전조 신호 학습, 불량 직전 이벤트 추적 |
| 필수 | PRESS 레시피/설정값 이력 | 압력·온도·진공 조건과 불량의 인과 분석 |

## 3. 있으면 모델 성능이 크게 좋아지는 데이터

| 우선순위 | 데이터 | 필요한 이유 |
|---|---|---|
| 권장 | LOT / PANEL / CYCLE 매핑키 | 품질-설비-알람-검사를 하나로 연결하는 핵심 키 |
| 권장 | 불량 상세 결과 | 불량 코드별 세분화 학습, 위치 기반 분석 |
| 권장 | 재검/재작업/스크랩 기록 | 실제 손실 및 반복 불량 패턴 분석 |
| 권장 | 유지보수/교정 이력 | 장비 열화와 교정 효과 반영 |
| 권장 | 작업자/교대/라인 정보 | 인적/운영 요인으로 인한 편차 분석 |
| 권장 | 원자재 lot / 공급사 정보 | 원자재 기인 품질 변동 추적 |
| 권장 | 환경 데이터(온도/습도/분진) | 외부 요인 보정 및 오탐 감소 |

## 4. 데이터 형식 권장사항

### 4-1. 공통 키

다음 키가 있으면 모든 테이블을 안정적으로 조인할 수 있습니다.

- `lot_id`
- `panel_id`
- `cycle_id`
- `machine_id`
- `process_id`
- `timestamp` 또는 `start_time` / `end_time`

### 4-2. 품질 데이터 권장 컬럼

- `date`
- `shift`
- `line_id`
- `product_code`
- `defect_code`
- `defect_name`
- `good_qty`
- `scrap_qty`
- `rework_qty`
- `ppm`

### 4-3. 설비 데이터 권장 컬럼

- `machine_id`
- `process_id`
- `state` (`run`, `stop`, `alarm`, `idle` 등)
- `start_time`
- `end_time`
- `downtime_min`
- `reason_code`
- `reason_name`

### 4-4. 알람 데이터 권장 컬럼

- `alarm_time`
- `alarm_code`
- `alarm_name`
- `severity`
- `machine_id`
- `duration_sec`
- `cleared_time`
- `ack_user`

## 5. 분석 관점에서 특히 중요한 요청 순서

1. **일별/주간/월별 품질 현황**
2. **설비 가동/비가동 및 공정별 가동율**
3. **PRESS 공정 알람 내역**
4. **PRESS 레시피/설정값 이력**
5. **LOT/PANEL/CYCLE 매핑키**
6. **유지보수/교정 이력**
7. **작업자/교대/라인 정보**

이 순서대로 받으면 먼저 경향 분석과 이상 구간 식별이 가능하고, 이후 인과 분석과 예측 모델 정확도를 빠르게 높일 수 있습니다.

## 6. 고객사에 보낼 요청 메시지 예시

```text
안녕하세요.

추가 분석을 위해 아래 데이터 제공을 요청드립니다.

1) 일별/주간/월별 품질 현황
2) 설비 가동/비가동 현황 및 공정별 가동율
3) PRESS 공정 알람 이력 및 설정값(레시피) 변화 이력
4) LOT / PANEL / CYCLE 매핑 키
5) 유지보수 / 교정 이력, 작업자 / 교대 정보

가능하시면 컬럼 정의서와 함께 전달 부탁드립니다.
감사합니다.
```

## 7. 기대 효과

- 일별/주간/월별 품질 추세로 이상 시점을 빠르게 찾을 수 있음
- 설비 가동율과 알람 로그로 불량의 선행 원인을 좁힐 수 있음
- PRESS 조건(압력/온도/진공)과 품질의 상관을 보다 직접적으로 학습할 수 있음
- 유지보수와 작업자/교대 정보를 반영하면 오탐을 줄이고 설명력을 높일 수 있음
